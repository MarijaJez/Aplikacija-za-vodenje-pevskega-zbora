# Aplikacija-za-vodenje-pevskega-zbora
Projekt pri predmetu Osnove podatkovnih baz.

Aplikacija bo vsebovala naslednje funkcionalnosti:
- pregled podatkov članov zbora
- beleženje pevskih vaj (potek dela, prisotnost) in nastopov
- baza programa zbora (za vsako pesem podatki o kategoriji, pdf z notami, ...)
- različni uporabniki imajo različne vloge in posledično pravice za dostop
- morda še kaj

V prihodnje bom dopolnila Readme datoteko z natančnejšo specifikacijo aplikacije.
